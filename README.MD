# 🚀 Portfólio Profissional - William Marcello Garcia Pereira

Um portfólio moderno, responsivo e futurista desenvolvido com **HTML5, CSS3 e JavaScript puro** (Vanilla JS).

## ✨ Características

- ✅ **Design Moderno**: Tema dark com glassmorphism e efeitos neon
- ✅ **100% Responsivo**: Desktop, tablet e mobile
- ✅ **Modo Claro/Escuro**: Alterna temas com persistência no localStorage
- ✅ **Sem Dependências**: Apenas HTML, CSS e JavaScript puro
- ✅ **Animações Suaves**: Transições elegantes e otimizadas
- ✅ **Acessível**: ARIA labels e navegação semântica
- ✅ **SEO Otimizado**: Meta tags e estrutura semântica
- ✅ **Rápido**: Performance otimizada
- ✅ **Menu Mobile**: Hamburger menu responsivo
- ✅ **Formulário Validado**: Validação em tempo real
